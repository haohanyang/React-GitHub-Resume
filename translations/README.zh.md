## 安装

为了设置本地开发环境，请遵循以下安装步骤

1. 克隆本项目到本机

```bash
git clone https://github.com/sabesansathananthan/React-GitHub-Resume.git
cd React-GitHub-Resume
```

2. 安装依赖包

   如果你用`npm`

   ```bash
   npm install
   ```

   或者

   如果你用`yarn`

   ```bash
   yarn
   ```

3. 运行开发服务器

   如果你用`npm`

   ```bash
   npm start
   ```

   或者

   如果你用`yarn`

   ```bash
   yarn start
   ```

4. 访问 <http://localhost:3000>
